# Lack Of Seed For Minting Update

## Summary
No summary provided.

## Details
## Mint Object Overview

In the Mint object, the `inner_hash` field provides a seed for minting new suifrens. This field should update after every mint with a new value.

## Code Snippet

```rust

    self: &mut Mint,

    birth_location: String,
    paid: &mut Coin<SUI>,
    ctx: &mut TxContext

    handle_payment(self, paid, ctx);
    let genes = hash(&bcs::to_bytes(&vector[
        self.inner_hash,
        bcs::to_bytes(&fresh_object_address(ctx)),
        bcs::to_bytes(clock)
    ]));
    let attributes = genes::get_attributes<T>(&self.id, &genes);
    suifrens::mint<T>(&mut self.id, 0, genes, attributes, clock, birth_location,
    â†’ option::some(self.mixing_limit), ctx)
}
```

## Remediation

Set the `self.inner_hash` value to `genes` at the end of `mint`.
