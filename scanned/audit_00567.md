# Unchangeable Dev Wallet Address

## Summary
No summary provided.

## Details
## Vault.move Structure

In `vault.move`, the `ProtocolFeeVault` structure stores the `FeePercentages` structure and the address of `dev_wallet`. Due to the lack of functionality for updating the address of `dev_wallet`, it remains immutable.

## Remediation

Implement the functionality for updating `dev_wallet`.

### vault.move RUST

```rust
public entry fun update_dev_wallet_address(
    protocol_fee_vault: &mut ProtocolFeeVault,
    new_dev_wallet_address: address,
    ctx: &mut TxContext
) {
    assert!(tx_context::sender(ctx) == protocol_fee_vault.dev_wallet, ENotDevWallet);
    protocol_fee_vault.dev_wallet = new_dev_wallet_address;
}
```
