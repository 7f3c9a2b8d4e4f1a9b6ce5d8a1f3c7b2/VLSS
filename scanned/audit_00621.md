## NFT Bridge Limitations

The current NFT Bridge explicitly does not support semi-fungible tokens.

## Use Cases

After discussion with ecosystem teams, it appears that there are some use cases. For example, we encountered one team which was building a ticketing platform where individual tickets would be fungible across sections, but each section would be represented as an individual NFT.

## Remediation

Consider adding explicit support for semi-fungible tokens. For example, Wormhole could implement a nonce to make semi-fungible tokens fully non-fungible.
