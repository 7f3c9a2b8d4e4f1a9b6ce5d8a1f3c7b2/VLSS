## Details
## Calculations Module Update

In the calculations module, `sui_to_afsui` and `afsui_to_sui` compute amounts based on the exchange rate. When the exchange rate becomes zero, an if statement is used to return the amount as zero. However, the if statement might be omitted because the following multiplication would result in zero.

## Remediation

- Remove the if statement that checks if `exchange_rate` is zero.
