## Code Optimization in `fixed.move`

In `fixed.move`'s `multiply_vec_by_fraction`, it is more efficient to store and reuse the value of `vector::length(vec)` instead of calling it multiple times.

## Code Snippet

```rust
public fun multiply_vec_by_fraction(vec: &mut vector<u64>, fraction: u64) {
    let index = 0;
    while (index < vector::length(vec)) {
        let num = vector::borrow_mut(vec, index);
        *num = multiply_by_fraction(*num, fraction);
        index = index + 1;
    };
}
```
