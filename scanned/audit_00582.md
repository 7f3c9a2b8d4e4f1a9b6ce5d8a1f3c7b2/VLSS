## Details
## vector_slice Function in utils.move

In `utils.move`, the `vector_slice` function should return the subslice of the vector, starting at the start index and ending at the end index. However, there is no check to ensure that `end` is higher than `start`; in this case, the function returns an empty vector.

## Function Declaration

```rust
public fun vector_slice<T: copy>(vec: &vector<T>, start: u64, end: u64): vector<T> {
    let vec_slice = vector::empty<T>();
    while (start < end) {
        vector::push_back(&mut vec_slice, *vector::borrow(vec, start));
        start = start + 1;
    };
    vec_slice
}
```

## Remediation

Ensure that the function aborts if `end < start` rather than returning an empty array.
