## Details
## Process Pending Stake Requests

The `actions::process_pending_stake_requests` function has a process for minting AFSUI and passing the amount to be minted through the `afsui_amount` variable when calling `mint_afsui`. Then, the amount of the minted AFSUI is fetched into the `minted_afsui_amount` variable via the `coin::value`. This step is redundant since `afsui_amount` essentially equals `minted_afsui_amount`.

## Code Snippet

```rust
fun process_pending_stake_requests(
    // iib. Mint the correct amount of `Coin<AFSUI>`.
    let afsui_coin = staked_sui_vault_state::mint_afsui(staked_sui_vault, safe, afsui_amount, ctx);
    let minted_afsui_amount = coin::value(&afsui_coin);
```

## Remediation

Remove `minted_afsui_amount` and replace it with `afsui_amount`.

## Patch

Fixed in `de5d36f` by removing `minted_afsui_amount`.
