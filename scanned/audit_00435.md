## Staked SUI Vault Update

The `create_staked_sui_vault` function calls `staked_sui_vault_state::create` to set parameters and utilizes multiple assert statements to validate the parameter values. Adding an assertion to check that `min_atomic_unstake_fee` is less than or equal to `max_atomic_unstake_fee` would be optimal.

## Remediation
