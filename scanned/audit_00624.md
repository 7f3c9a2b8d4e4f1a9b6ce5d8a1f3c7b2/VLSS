# Unused BankInfo Field

## Summary
No summary provided.

## Details
## BankInfo Structure and Functionality

The `is_listed` field in the `BankInfo` struct determines whether borrowing is allowed from the bank.

## BankInfo Structure

```rust
struct BankInfo has store {
    is_listed: bool, // whether borrowing is allowed.
    index: u8, // Reverse look up index for this bank.
    total_share: u64 // The total debt share count across all open
                     // positions.
}
```

However, the current bank module provides no way to change it from the default value of `true` set in `init_bank`.

## Initialize Bank Function

```rust
/// @dev Add a new bank to the ecosystem.
public entry fun init_bank<CoinType>(acc: &signer) acquires MetaData {
    // ...
    // 1. add bank info
    let bank = BankInfo {
        is_listed: true,
        index: (index as u8),
        // total_debt: 0,
        total_share: 0
    };
}
```

## Remediation

Create admin-gated functions to modify `BankInfo::is_listed`.
