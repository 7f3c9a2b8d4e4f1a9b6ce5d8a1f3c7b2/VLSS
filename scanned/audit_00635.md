## Details
## DGEN Coin Module

The file `dgen/sources/dgen.move` implements the DGEN coin module. The function that initializes the coin and mints a `TOTAL_SUPPLY` contains a small error in the description.

```rust
/// Initializes DGEN coin, making total supply premint for owner.
/// * `dgen_owner` - deployer of the module.
public entry fun initialize(dgen_admin: &signer) {
[...]
```

The deployer of the module is `dgen_admin`.

## Remediation

Replace `dgen_owner` with `dgen_admin`.
