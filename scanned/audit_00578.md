# Mismatched Constant Usage

## Summary
No summary provided.

## Details
## Permission Management

In `permission.move`, `get_admin_permission` and `get_coin_permission` use a default policy name `RESERVED_DEFAULT_PERMISSION_NAME`, while `init_default_policy` uses the constant `DEFAULT_POLICY_NAME`. This will cause issues if the values of those constants are changed in the future.

## Remediation

Utilize the constant `DEFAULT_POLICY_NAME` in `get_admin_permission` and `get_coin_permission`.
